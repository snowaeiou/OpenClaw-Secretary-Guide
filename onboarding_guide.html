OpenClaw 秘書實戰手冊

主題： 將 AI 龍蝦變成你的最強分身

版本： 1.0 (2026年2月)

作為一名專業秘書，你嘅時間係用嚟做決策同溝通，而唔係處理煩瑣嘅數據搬運。OpenClaw（我哋叫佢做「龍蝦」）就係你嘅 24 小時數碼分身，幫你執行所有枯燥但重要嘅任務。

第一章：馴服龍蝦的第一步（基礎設定）

1. 身份訓練： 話俾龍蝦聽你係邊個。喺 USER.md 入面寫低你嘅習慣、邊個係你老闆、你鍾意點樣排版。

2. 角色分工： 透過 Dashboard 監控唔同嘅 Agent。你可以有一個 Agent 專門搞 Payroll，一個專門搞 Coding。

圖 1：OpenClaw Gateway Dashboard - 喺呢度你可以睇到有幾多個「分身」正在幫你開工。

第三章：點樣落指令（Prompting 技巧）

龍蝦原則： 指令愈具體，結果愈完美。
❌ 錯誤示範 (Before)
「幫我執下啲 Email。」

結果：龍蝦唔知由邊度執起，可能會問返你轉頭。
✅ 專業指令 (After)
「幫我搵返過去 7 日所有嚟自 Apple 嘅收據，將標題同金額列個表俾我。」

結果：龍蝦即刻列出清單，甚至可以問你要唔要整成 PDF。

第四章：常見問題 Q&A

Q: 點解龍蝦好似唔記得咗我之前講過嘅嘢？
A: 龍蝦每個對話（Session）都係新嘅。如果重要嘅嘢，請叫佢「記落 MEMORY.md 度」，咁佢下次開工就會讀返出嚟。

Q: 指令出咗 Error 點算？
A: 唔使驚，通常係 Token 用完或者 API 暫時斷線。你可以叫佢「再試一次」或者「檢查下個 config」。
